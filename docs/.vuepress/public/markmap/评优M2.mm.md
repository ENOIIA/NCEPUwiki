# 评优M2
## （一）学习成绩（100分） $\times$ 88%
  - 参评课程为测评学年应修本专业的必修课
  - 成绩按课程**首次考试**实际成绩计算
  - 学校不鼓励缓考，确定需要缓考的，经**教务处批准**后按缓考成绩计算
  - 重修课程须在重修学年学分绩计算中予以体现
  - 学年平均学分绩以**教务处**实际提供的为准
  - $学年平均学分绩=\cfrac{\sum(课程学期成绩 \times 取得的课程学分)}{\sum所学课程学分}$
